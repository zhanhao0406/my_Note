# Wrokspace 是什么？
---
* scr:       source space;  -> 代码空间，人为放置。
* build:     build space;   -> 编译空间，编译过程中产生的中间文件。
* install:   install space; -> 安装空间，编译得到的可执行文件和脚本所在地。
* log:       log sapce;     -> 日志空间，编译和运行中产生的日志信息。
---

## 创建工作空间   
创建工作空间，然后在 /src 中放入代码。  
```c++
$ mkdir -p ~/dev_ws/src
$ cd ~/dev_ws/src
$ git clone https://gitee.com/guyuehome/ros2_21_tutorials.git
```
## 安装依赖 （可手动安装，也可使用rosdep（中文地区鱼香rosdepc）工具自动安装）
```c++
$ sudo apt install -y python3-pip
$ sudo pip3 install rosdepc
$ sudo rosdepc init
$ rosdepc update
$ cd .. 
$ rosdepc install -i --from-path src --rosdistro humble -y
```

## 编译工作空间 (ROS2不再使用catkin_make / build ；现在使用<mark>colcon build</mark>)
```c++
$ sudo apt install python3-colcon-ros
$ cd ~/dev_ws/ #只能在主工作空间colcon build
$ colcon build
```

## 设置环境变量

`$ source install/local_setup.sh # 仅在当前终端生效，每次都要使用。`  
**OR**  
`$ echo " source ~/dev_ws/install/local_setup.sh" >> ~/.bashrc # 所有终端均生效，需要打开bashrc文件，添加source 路径`
