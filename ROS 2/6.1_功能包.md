# 功能包

## 创建功能包
* **pkg** -》功能包相关功能
* **create** -》创建
* **--build-type** -》决定是哪种的功能包？c++是*ament_cmake* 还是 python是*ament_python*
* **package_name** -》功能包的名字
---
在src中创建指令：  
```c++
$ cd ~/dev_ws/src
$ ros2 pkg create --build-type ament_cmake learning_pkg_c  # C++
$ ros2 pkg create --build-type ament_python learning_pkg_python  # Python
```

## 编译功能包
```
$ cd ~/dev_ws/src
$ colcon build   # 编译工作空间所有功能包
$ source install/local_setup.bash  #如果已经在~/.bashrc 中添加过路径，就可以不用这一步
```
## 功能包结构
C++中必然存在：  
* **package.xml** -》 包含功能包的版权描述，和各种依赖的声明
* **CMakeLists.txt** -》C++代码编译规则
