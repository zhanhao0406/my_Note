# 分布式

需要有另一个“树莓派”，与电脑ubuntu在同一个网络下进行通信。  
`ifconfig   #获取ip地址`   
`ping   #跟ip地址确保ping 通`  
之后：  
`ssh  #后面跟客户端名称，回车后输入密码，即可进入新的客户端进行指令操作`


正常来说，不需要进行网络配置，只需要确保两个在相同局域网，就可以运行：   
`ros2 run demo_nodes_py talker`  
`ros2 run demo_nodes_py listener`   

也可以使用分组设置，确保信息安全：   
`sudo gedit ~/.bashrc`  
打开后添加：  
`export ROS_DOMAIN_ID=     #=任意一个数字，两个终端都需要设置为相同的domain`  
然后  
`source ~/.bashrc`  
然后再进行测试即可完成通讯。
