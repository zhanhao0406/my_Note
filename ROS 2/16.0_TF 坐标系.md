# TF 坐标系 (均为python代码)

Base frame, World frame。   
海龟跟随Example：  
```text
sudo apt install ros-humble-turtle-tf2-py ros-humble-tf2-tools
sudo pip3 install transforms3d
ros2 launch turtle_tf2_py turtle_tf2_demo.launch.py
ros2 run turtlesim turtle_teleop_key
ros2 run tf2_tools view_frames    #PDF展示节点图
ros2 run tf2_ros tf2_echo turtle2 turtle1  # 给出坐标之间的数据区别， turtle2相较于turtle1
ros2 run rviz2 rviz2 -d $(ros2 pkg prefix --share turtle_tf2_py)/rviz/turtle_rviz.rviz  # RVIZ中可视化
```
