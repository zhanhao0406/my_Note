## 命令行帮助信息
与ros1 相比较  
ros2 node == rosnode (in Ros1)  [list ... pub ... echo ...]   
ros2 topic == rostopic (in Ros1)  [list ... pub ... echo ...]   


## turtlesim 仿真举例。  
* 创建小乌龟 -> `ros2 run turtlesim turtlesim_node`
* 键盘控制 -> `ros2 run turtlesim turtle_teleop_key`

查看节点之类的 ->  `ros2 node list` 或者 `ros2 node info /turtlesim` 或者 `ros2 topic list` 或者 `ros2 topic echo /turtle1/pose`   

* 发布话题消息(指令移动) -> `ros2 topic pub --rate 1 /turtle1/cmd_vel geometry_msgs/msg/Twist "{linear: {x: 2.0, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: 1.8}}"`
* 发布服务请求(添加另外的乌龟) ->  `ros2 service call /spawn turtlesim/srv/Spawn "{x: 2, y: 2, theta: 0.2, name: ''}"`

然后可以用 `ros2 topic list` 查看新的话题有没有添加。
* 发布服务请求（对另外添加的乌龟<mark>turtle2</mark>）-> `ros2 topic pub --rate 1 /turtle2/cmd_vel geometry_msgs/msg/Twist "{linear: {x: 2.0, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: 1.8}}"`
---
* 录制bag（速度） -> `ros2 bag record /turtle1/cmd_vel`
* 播放bag -> `ros2 bag play *文件名称*`
